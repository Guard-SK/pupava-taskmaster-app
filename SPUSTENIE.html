Spustenie a užitočné príkazy (macOS)

Návod na zapnutie, vypnutie a riešenie oprávnení pre aplikáciu Pupava Taskmaster.

🚀 Ako zapnúť a vypnúť hru

1. Zapínanie

  1. Spusti program Visual Studio Code a uisti sa, že máš otvorený priečinok s hrou.
  2. V hornom menu programu klikni na Terminal a zvoľ New Terminal (dole sa otvorí čierne okno s blikajúcim kurzorom).
  3. Klikni myšou do tohto čierneho okna, napíš nasledujúci príkaz a stlač kláves Enter: 
    npm run dev
  4. Počkaj 2–3 sekundy, kým sa v okne nezobrazia internetové adresy. Hra je v tomto momente úspešne zapnutá a beží na pozadí.

2. Ako hru otvoriť v prehliadači

Aplikácia má 3 hlavné časti, ktoré sa otvárajú na rôznych adresách:

  • Pre teba (na počítači):
    • Administrátorská konzola (moderátor): http://localhost:5173/admin (tu ovládaš celú hru, pridávaš slajdy a priraďuješ body)
    • Premietacia obrazovka (projektor): http://localhost:5173/projector (túto stránku si presuň na projektor pre divákov – zobrazuje priebežné poradie a výsledky)
  • Pre deti:
    • Hlasovacia stránka pre tímy: http://[IP_ADRESA_MACu]:5173/client (alebo len základná adresa bez koncovky: http://[IP_ADRESA_MACu]:5173)
    • Poznámka: Deti sa musia najprv pripojiť na tú istú Wi-Fi sieť, ako tvoj počítač. Adresu s reálnymi číslami (IP adresou tvojho Macu) uvidíš v termináli pod riadkom Network:.

3. Ako hru vypnúť

Keď chceš program ukončiť:

  1. Klikni myšou do čierneho okna terminálu vo VS Code.
  2. Stlač na klávesnici naraz klávesy Ctrl + C (pozor, na Macu je to v termináli naozaj kláves Control / Ctrl, nie Command/Cmd).
  3. Servery sa vypnú a môžeš bezpečne zavrieť celý program Visual Studio Code.

🛠️ Užitočné príkazy (macOS)

Tieto príkazy pomáhajú vyriešiť problémy s inštaláciou, oprávneniami alebo zistením sieťovej IP adresy.

1. Inštalácia modulov (pri problémoch s oprávneniami)

Ak bežný príkaz npm install vypíše chybu ohľadom oprávnení (permission denied), použi inštaláciu s povolením pre nebezpečné práva:

sudo npm install --unsafe-perm

(Bude potrebné zadať heslo k tvojmu Macu – pri písaní sa heslo z bezpečnostných dôvodov nezobrazuje, len ho napíš a stlač Enter).*

2. Oprava vlastníctva priečinka (Permission Denied)

Ak server nedokáže zapisovať súbory (napr. ukladať stav hry do state.json) alebo hlási chyby prístupu, nastav seba ako vlastníka celého priečinka projektu:

sudo chown -R $(whoami) /Users/tomassuchanek/pupava-taskmaster

(Tento príkaz rekurzívne priradí vlastníctvo priečinka tvojmu aktuálnemu macOS užívateľovi).*

💡 Alternatívny rýchly variant:

Ak si v termináli priamo v priečinku projektu, stačí použiť bodku . namiesto celej cesty:

sudo chown -R $(whoami) .

3. Rýchle zistenie lokálnej IP adresy

Pre zistenie IP adresy tvojho Macu v lokálnej Wi-Fi sieti (aby sa vedeli pripojiť deti z mobilov) spusti príkaz:

ipconfig getifaddr en0

Príkaz ti okamžite vypíše tvoju lokálnu IP adresu (napr. 192.168.1.52), ktorú potom dosadíš do odkazu pre mobily: http://[IP_ADRESA]:5173.
